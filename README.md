# Менеджер задач

Учебный проект для практической работы 13 по теме «Создание простого приложения на React».

Приложение «Менеджер задач» позволяет добавлять задачи, отмечать их выполненными, удалять, фильтровать список и сохранять данные в `localStorage`, чтобы задачи восстанавливались после обновления страницы.

Репозиторий: https://github.com/git21git/practice_13_react

GitHub Pages: https://git21git.github.io/practice_13_react/

## Функции

- Добавление новой задачи через форму.
- Отметка задачи как выполненной.
- Удаление задачи.
- Фильтрация задач: все, активные, выполненные.
- Счетчик оставшихся активных задач.
- Сохранение задач в `localStorage`.
- Восстановление задач после обновления страницы.

## Технологии

- React
- JavaScript
